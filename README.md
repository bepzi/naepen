<h1 align="center">Naepen</h1>

A toy cross-platform polyphonic synthesizer.

### Building

On Arch Linux:

```bash
sudo pacman -S juce
git clone <repo_url.git> <repo> && cd <repo>
cd Builds/LinuxMakefile  # Alternatively, build in CLion
make -j
```
